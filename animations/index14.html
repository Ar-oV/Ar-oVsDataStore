CSS Transform

translate (x, y)

translate

translate - сдвигает элемент на новое место, перемещая относительно изначального положения объекта вправо и вниз, используя координаты X и Y. Записывается следующим образом (x,y)

scale (x, y)

scale

scale - масштабирует элементы, делая их больше или меньше. По умолчанию значения указаны как еденица (1) - это реальный размер объекта. Первое значение масштабирует элемент по ширине, второе - по высоте. Отрицательные значения отображают элемент зеркально.

rotate (угол+deg)

rotate

rotate - поворачивает элементы на заданное количество градусов,отрицательные значения поворачивают элемент против часовой стрелки, положительные - по часовой стрелке, записываются значения как градус поворота и deg (x,y)

skew (x-угол+deg, y-угол+deg)

skew

skew - деформирует (искажает) стороны объекта относительно координатных осей. Если указано одно значение, второе будет определено браузером автмоатически. Записываются значения как градуус поворота и Deg

matrix (a,c,b,d,x, y)

matrix

matrix - позволяет объеденить несколько функций 2D-трансформаций в одной. В качестве трансформации доппустимы поворот, масштабирование, наклон (деформация) и изменение положения. Свойства указываются без едениц измерения.

transform origin

transform origin - позволяет сместить центр трансформации, относительно которого происходит изменение опложения/размера/формы элемента

perspective, perspective-origin

perspective, perspective-origin

perspective, perspective-origin - установка глубины и смена точки начала координат (x,y)

translate3D

translate3D

translate3D - по аналогии со своим 2D братом это значение задает перемещение элемента, но, в 3D-пространстве. Помимо оси X и Y добавляется ось Z.

scale3D

scale3D

scale3D - задает трехмерное масштабирование по одной из осей X,Y,Z

rotate3D

rotate3D

rotate3D - Вращает элемент относительно трех осей. Элемент поворачивается под углом, задаваемым последним параметром относительно вектора направления X,Y,Z

matrix3D

matrix3D

matrix3D - задает трехмерное преобразование как однородную матрицу размером 4ч4 сс шестнадцатью значениями в столбцах (x,y)
